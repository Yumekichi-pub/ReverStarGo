📒 ReverStarGo イベント台帳

このファイルは、ReverStarGo の本体プログラムが発火する 全てのイベント（観察可能な動作・状態変化）を 1 箇所にまとめた一次資料です。

使い方

  • 編集の自由度: このファイルは Markdown なので、いつでも任意のエディタで開いて編集できます
  • 追加方法: 新しいイベントが見つかったら、該当カテゴリの表に 1 行追加するだけ
  • 削除方法: 行を削除するか、行頭に ~~ を付けて打ち消し線にする
  • チェッカからの参照: 同フォルダの event_check.html を起動すると、このファイルを fetch で読み込み、各 ID を自動でチェック対象にします
  • ID 命名規則: カテゴリ略号_連番 の形式（例: MOV_001, SND_003）
